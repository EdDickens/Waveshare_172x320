This basic configuration will display the Time and Date from Home Assistant in Landscape mode using ESPHome LVGL.

The "display" section has alternate dimensions and tramsform commented out if you want to use Portrait mode.
The "offset_height" and "offset_width" are needed to center the objects on the display.
